# EasyList App

A simple ToDo list built on NodeJS, MongoDB and AngularJS


## How to run

First of all, you will need a local instance of MongoDB and run it with :

```
$> mongod
```

Install needed libs through npm with :

```
$> npm install
```

Finally run the server:

```
$> node app.js
```

And enjoy the app at [http://localhost:8080](http://localhost:8080)
